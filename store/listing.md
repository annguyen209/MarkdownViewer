# Microsoft Store Listing - MarkdownViewer

Short description:
A fast, accessible Markdown and text file viewer for Windows with live preview, syntax highlighting, and a clean, customizable UI.

Full description:
MarkdownViewer is a lightweight desktop application for viewing Markdown and other text files. Features include a split editor/preview view, syntax highlighting for code blocks, drag & drop support, recent file history, theme toggle (light/dark), font sizing controls, and export-friendly rendering suitable for quick reading and sharing.

Highlights:
- Live Markdown preview with syntax highlighting (highlight.js)
- Drag & drop and File->Open support for local files
- Recent files list for quick access
- Theme and font controls for accessibility

What's new in this release:
- Initial release with basic editor + rendered preview, file opening, and theming options.

Privacy and permissions:
- The app reads local files selected by the user. No data is sent externally.
- No sign-in is required.
